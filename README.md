🥜 Peanut Quality Classification

📌 Introduction

Quality assessment of peanuts is a crucial step in the agricultural and food processing industries,
as it directly impacts product value, consumer safety, and market standards. 
Traditional methods of quality inspection are often manual, time-consuming, and prone to human error.
This project presents an automated Peanut Quality Classification system using Machine Learning and Deep Learning techniques. 
The model is trained on image data to accurately classify peanuts based on their quality. By leveraging computer vision and 
neural networks, the system minimizes human intervention and improves efficiency.
The proposed solution aims to provide a reliable, scalable, and efficient approach for real-time quality analysis,
which can be beneficial for farmers, food industries, and supply chain management.

🚀 Features

- Image-based peanut quality classification
- Deep Learning model using TensorFlow/Keras
- Predicts quality from new input images
- Reduces manual inspection effort

🛠️ Tech Stack

- Python
- TensorFlow / Keras
- NumPy
- OpenCV
- Matplotlib

📂 Project Structure

dataset/              # Training images
train.py              # Model training script
predict.py            # Prediction script
peanut_model.h5       # Trained model
test.jpg              # Sample test image
README.md             # Project documentation


⚙️ How to Run

1. Install required libraries:
   pip install tensorflow numpy opencv-python matplotlib

2. Train the model:
   python train.py

3. Run prediction:
   python predict.py

📊 Use Case

This system can be used in:

- Peanut processing industries
- Agricultural quality control
- Automated sorting systems
